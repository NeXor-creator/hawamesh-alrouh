# hawamesh-alrouh
"هوامش الروح" هو مساحتي الرقمية الهادئة التي أكرسها للشعر والأدب، بعيدًا عن ضجيج العالم الرقمي، حيث تصبح القصيدة محور التجربة بكل نقائها. يأتيك الموقع بواجهة نظيفة بلون كريمي هادئ (#FAF3E0) وخطوط عربية أنيقة (Cairo للعناوين وAmiri للنصوص)، ليقدم محتواه عبر أقسام متمايزة: "همسات الروح" للقصائد التأملية القصيرة، "خطوط الروح" للأشعار الطويلة أو السردي
